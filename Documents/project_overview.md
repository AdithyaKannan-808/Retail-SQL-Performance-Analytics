#  Project Overview

This project delivers an end-to-end **SQL analytics workflow** for retail sales performance.  
It integrates transactional orders, return records, and salesperson-region mappings into a unified analytical model, enabling monthly performance measurement, target modelling, and ranking.

The project demonstrates how SQL alone can be used to build a complete analytics pipelineâ€”similar to how businesses track sales efficiency, profitability, and target attainment.

---

##  Objective

To transform raw retail sales data into actionable insights by:

- Integrating multiple datasets using SQL joins  
- Generating monthly sales & profit actuals  
- Creating synthetic monthly targets (+3% growth model)  
- Comparing actual vs target performance  
- Classifying salespeople (Overachiever / On Target / Underperformer)  
- Ranking performance using window functions  
- Extracting KPIs and return-rate insights
